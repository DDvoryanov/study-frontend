# Картинки

## Краткая информаци

```html
<img src="cover.png" alt="HTML tutorial"/>
```

Для создания ссылок на другие страницы используется тег **`<img>`**

- **`<src>`** путь к картинке
- **`<alt>`** текст в случае отсутвия картинки
